## Overview

The **Student Attendance-System** is a modern mobile solution designed to digitize and secure the attendance process at Universitas Bani Saleh. By leveraging GPS technology, it ensures that students are physically present on campus when marking their attendance.

## Key Features

- **GPS Verification**: Prevents proxy attendance by requiring students to be within the campus geofence.
- **Seamless UX**: A clean, intuitive interface that allows students to check in with a single tap.
- **Real-time Reporting**: Instant updates for faculty and administration regarding student presence.
- **Secure Authentication**: Robust login system to protect student data.

## Technical Stack

- **Platform**: Mobile (Flutter / React Native)
- **Backend**: Firebase / Node.js
- **Services**: Geolocation API, Authentication

## Outcome

The design and prototype successfully addressed common issues with manual attendance taking, proposing a streamlined, fraud-resistant digital alternative for the university.
