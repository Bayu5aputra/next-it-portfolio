## Overview

**ShowPro** is a dealership web profile application developed to demonstrate proficiency with the **CodeIgniter 4** framework. It functions as a complete product showcase and management system, suitable for small to medium-sized businesses.

## Key Features

- **Product Catalog**: A public-facing showcase of vehicles/products with detailed views.
- **Admin Dashboard**: A secure backend for adding, editing, and removing listings.
- **User Management**: Authentication system with Login, Logout, and Role management.
- **Responsive Design**: Built with **Bootstrap** to ensure accessibility on all devices.

## Technical Stack

- **Framework**: CodeIgniter 4 (PHP)
- **Database**: MySQL
- **Frontend**: HTML5, CSS3, JavaScript, jQuery, Bootstrap

## Outcome

This project served as a comprehensive proof of capability in MVC (Model-View-Controller) architecture, database management, and secure web application development using PHP.
