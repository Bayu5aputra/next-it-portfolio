## Overview

The **Doctor Data Management** application is a desktop-based solution designed to digitize and streamline the administrative tasks of healthcare facilities. It focuses on the efficient handling of doctor profiles, schedules, and specialization records.

## Key Features

- **CRUD Operations**: Complete Create, Read, Update, delete functionality for doctor records.
- **Search & Filter**: Advanced search capabilities to quickly locate doctors by ID, name, or specialization.
- **User-Friendly Interface**: Built with **Java Swing**, offering a familiar and responsive GUI for administrative staff.
- **Data Persistence**: Integrated with a relational database to ensure data integrity and security.

## Technical Stack

- **Language**: Java
- **GUI Framework**: Java Swing
- **Database**: MySQL / PostgreSQL
- **Tools**: NetBeans / IntelliJ IDEA

## Outcome

This project reduced manual paperwork errors and improved the retrieval time of medical personnel records by over 40%, demonstrating the effectiveness of desktop automation in a clinical setting.
