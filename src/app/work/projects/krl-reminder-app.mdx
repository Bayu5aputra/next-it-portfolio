## Overview

The **KRL Reminder App** addresses a common problem for daily commuters: falling asleep or getting distracted and missing their stop. This mobile application uses GPS tracking to trigger an alarm when the user approaches their destination station.

## Key Features

- **Location-Based Alerts**: Triggers an alarm and vibration when the device enters a geofenced area around the destination station.
- **Commuter-Centric Design**: Specifically tailored for the KRL Commuterline network in Indonesia.
- **Sleep Mode**: Allows users to nap with confidence, knowing they will be woken up before arrival.
- **Offline Capable**: Core features function reliably even with spotty internet connection on the train.

## Technical Stack

- **Platform**: Android
- **Language**: Kotlin / Java / Dart (Flutter)
- **Services**: GPS / Geolocation API, Notification Manager

## Outcome

The app provides peace of mind for tired commuters, acting as a reliable digital assistant that ensures they always arrive at their destination on time.
