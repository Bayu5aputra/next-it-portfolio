## Overview

**Live Content Digital Hub** is a robust web application tailored for managing and broadcasting digital content across an organization. It serves as a central repository for media assets and provides a seamless playback experience for end-users.

## Key Features

- **Centralized CMS**: A powerful Content Management System allowing admins to upload, organize, and schedule content.
- **Auto-Play Viewer**: A dedicated frontend interface that automatically cycles through scheduled content (videos, images, announcements) without user intervention.
- **Multi-Level Access**: Role-based access control (RBAC) to ensure different departments can manage their own content securely.
- **Laravel Framework**: Built on **Laravel**, ensuring scalability, security, and maintainability.

## Technical Stack

- **Framework**: Laravel (PHP)
- **Database**: MySQL
- **Frontend**: Bootstrap / Tailwind CSS
- **Deployment**: Linux Server

## Outcome

This platform streamlined the distribution of internal communications and visual announcements, replacing manual USB-drive updates with a fully networked, cloud-managed solution.
