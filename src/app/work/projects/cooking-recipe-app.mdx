## Overview

The **Cooking Recipe App** is a digital culinary companion built with **Flutter**. It allows food enthusiasts to discover, search for, and save recipes. The application focuses on a smooth user experience and high visual appeal to make cooking more accessible.

## Key Features

- **Recipe Search**: Fast filtering to find dishes by name or ingredient.
- **Culinary Tips**: A dedicated section for cooking advice and techniques.
- **User Profiles**: Login system allowing users to save their preferences (expandable).
- **Cross-Platform**: Deployable to both iOS and Android from a single codebase.

## Technical Stack

- **Framework**: Flutter (Dart)
- **State Management**: Provider / Riverpod
- **API**: REST API Integration (for recipe data)

## Outcome

The project demonstrated the efficiency of Flutter for building attractive, high-performance mobile applications with complex UI requirements.
