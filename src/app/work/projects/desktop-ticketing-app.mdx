## Overview

The **Java Ticketing Management Application** is a specialized desktop software designed for high-volume environments like amusement parks. It provides a reliable interface for staff to issue, modify, and track ticket sales securely without relying on a web browser.

## Key Features

- **Comprehensive CRUD**: Full control over ticket order data (Create, Read, Update, Delete).
- **Instant Search**: Rapid filtering mechanism to retrieve booking details by customer name or ID.
- **Data Persistence**: Connected to a local **MySQL** database for reliable record-keeping.
- **Native Performance**: Fast execution and low resource usage typical of native Java applications.

## Technical Stack

- **Language**: Java
- **GUI Framework**: Java Swing / AWT
- **Database**: MySQL
- **Connectivity**: JDBC

## Outcome

This application provided a stable and efficient solution for on-site ticketing operations, minimizing transaction times and eliminating dependence on internet connectivity.
