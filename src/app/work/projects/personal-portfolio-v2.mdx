## Overview

**Personal Portfolio v2** represents the evolution of my digital presence. Moving away from static templates, this version focuses on dynamic content loading, high performance, and a refined minimalist aesthetic.

## Key Features

- **Minimalist Design**: A clean, whitespace-heavy layout that puts the focus entirely on content and typography.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop devices using CSS Grid and Flexbox.
- **Serverless Backend**: Integrated with **Firebase** for hosting and real-time database capabilities if needed.
- **Performance**: Achieved high Lighthouse scores through code splitting and optimized asset loading.

## Technical Stack

- **Frontend**: React.js
- **Styling**: Styled Components / CSS Modules
- **Hosting**: Firebase / Vercel
- **Animations**: Framer Motion

## Outcome

This portfolio served as a successful demonstration of modern frontend practices, resulting in increased engagement and a professional showcase for potential employers and clients.
